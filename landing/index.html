Long-term memory

Agents with schedules

Create custom AI assistants with their own instructions, tools, and cron schedules. Run tasks automatically and deliver results to any platform.

Attach images to any message. Works with all vision-capable models — local (LLaVA, Llama 3.2 Vision) and external (GPT-4o, Claude, Gemini).

Get started

Up and running in minutes

Choose the installation method that suits you best.

Option 1 — Desktop app

Download

Grab the installer for your platform from the GitHub releases page. Includes Ollama — no separate install needed.

First launch

A setup screen appears. Choose whether to also add the CLI to your PATH. The AI server starts automatically.

Start chatting

Your browser opens to localhost:1616. Create your account, pull a model, and start your first conversation.

Option 2 — Docker (NAS / server / headless)

Ollama is bundled inside the image — single container, no sidecar needed.

docker run -d \
  -p 1616:1616 \
  -v enzo-data:/app/data \
  ghcr.io/eranabir/enzo-ai:latest

Then open http://your-server-ip:1616 from any browser on your network.
